What is Homebrew?

Homebrew usually refers to software that is not authorized by Nintendo. Installing a homebrew exploit will allow you to run homebrew games, use tools like save editing and backup, and use emulators for various older systems.

Homebrew will be 100% free using the DSi Browser Application or the DSi Flipnote Application, however that has not been released yet.
Other DSi homebrew exploits can be installed via the Sudoku application and with a hardware mod.

What is RocketLauncher?

RocketLauncher is a DSi exploit on the firmware version 1.4. It will only ever work on 1.4. Firmwares below or above 1.4 do not work.
This is because on the version 1.4, Nintendo forgot to reimplement RSA checks on the DS card whitelist.

The exploit requires a flashcard with a flashable NAND, much like NTRBoot. A compatibility list will be released at a later date.
This requires an exisitng DSiWare exploit or a hardware mod to install. This includes downgrading and installing the modified DS card whitelist which is needed for RocketLauncher to work.

What is HiyaCFW?

HiyaCFW is the first CFW (Custom Firmware) for the DSi. It runs using RocketLauncher, meaning that only version 1.4 is compatible.
The custom firmware includes RSA and signature patchers, has a custom launcher, and is region-free. You will be able to install free DSiWare as well as custom applications and homebrew into the DSi Menu.
