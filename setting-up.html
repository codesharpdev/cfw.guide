Vita Guide

Setting Up HENkaku

This step will setup HENkaku Ensō for use on your PS Vita

Disabling Safe Mode

  1. Open the Settings app
  2. Select "HENkaku Settings"
  3. Check "Enable Unsafe Homebrew"

Download Ensō

  1. Open the Web Browser on your PS Vita
  2. Visit this page on your Vita and click this link to download HENkaku Ensō
  3. Download the file
  4. Go back to the Home Menu and open MolecularShell
  5. Navigate to "ux0:video/aa
    • Not necessarily "aa", it could be any two letters, e.g. "hg", "cf", etc
  6. Rename "enso.mp4" to "enso.vpk"
  7. Select the file and install it

Proceed to Installing HENkaku Ensō
