Vita Guide

Setting Up HENkaku

This step will setup HENkaku Homebrew on your PS Vita

Disabling Safe Mode

  • Open the Settings app
  • Select "HENkaku Settings"
  • Check "Enable Unsafe Homebrew"

    • Download Enabler

  • Open the Web Browser on your PS Vita
  • Visit this page on your Vita and click this link to download Download Enabler
  • Download the file
  • Go back to the Home Menu and open MolecularShell
  • Navigate to "ux0:video/aa
  • Not necessarily "aa", it could be any two letters, e.g. hg, cf, etc
  • Rename "download_enabler_v3.mp4" to "download_enabler_v3.suprx"
  • Press Triangle and select "Copy" to copy this file
  • Navigate to "ux0:tai" and paste it there
  • Open "config.txt"
  • Add the following to the file:
    *main
    ux0:tai/download_enabler_v3.suprx
  • Exit and save changes
  • Reboot your console

    • Installing Homebrew

  • Open this page on your PS Vita Browser
  • Download all of the following files:
  • VitaShell
  • Vita Homebrew Browser
  • HENkaku Ensō
  • Open MolecularShell
  • Navigate to ux0:download
  • Install all three ".vpk" files
  • If any fail to install, try installing VitaShell first and install them with that
  • These three applications should now be installed on your Home Menu

    • Proceed to Installing HENkaku Ensō
