How do I block updates without losing access to the eShop?

It's pretty simple really, just requires a bit of work.

WARNING!

FTPiiU_everywhere is potentially dangerous if used incorrectly. Follow this guide carefully.
While this guide will not brick your Wii U, this tool has complete access to all sysNAND partitions.
Deletion of files unspecified in this guide may result in an un-recoverable brick!

Requirements:

  • FTPiiU_everywhere
  • Homebrew Launcher Access
  • FTP Client

Instructions:

  1. Place ftpiiu_everywhere.elf into the directory "SD:wiiu/install/apps/ftpiiu_everywhere" on your SD Card
  2. Launch the Homebrew Launcher from your entrypoint of choice
  3. Launch FTPiiU_everywhere
  4. Open your FTP client and connect to the Wii U with the IP address on screen
  5. Navigate to the directory "NAND:storage_mlc/sys"
  6. Delete the folder "update"
  7. Exit out of FTPiiU_everywhere on your Wii U and go back to the Home Menu
  8. Launch System Settings and go to System Update
  9. Attempt to update your system
    • If it immediately fails, you did it right!
    • If it even makes the smallest bit of progress, CANCEL IT
    • Retry steps 2-6 until it works (it took me a few times)

That's great, but the Wii U still asks for updates when I go online! How can I bypass this!!

This is, too, pretty simple. You just need to install the 5.5.2 VersionData title.

Requirements:

  • Your corresponding zip for each region
    • Go to "pastebin.com/u/username"
    • Replace "username" with the site's address (don't include the "." or the "/wiiu")
    • Click on the paste titled "Wii U Spoofing"
  • Homebrew Launcher Access

Instructions:

  1. Delete the directory "SD:install" if it exists
  2. Copy the contents of your ".zip" file to the root of your SD card
  3. Overwrite files if prompted
  4. Launch the Homebrew Launcher from your entrypoint of choice
  5. Launch WUPInstaller
  6. Press A to install to system memory
  7. After installation, check system settings
    • If it says "5.5.2", you did it correctly!
    • If it does not, redo these steps and see where you went wrong

I can now use games online, but how do I use the eShop?

Again, quite simple. You just need to run some homebrew before using it.

Requirements:

  • nnupatcher
  • Homebrew Launcher Access

Instructions:

  1. Copy the contents of nnupatcher.zip into the root of your SD card
  2. Launch the Homebrew Launcher from your entrypoint of choice
  3. Launch nnupatcher
  4. Go back to the Home Menu and launch the eShop
    • You should be able to get into the eShop without updating!
    • If it asks for an update, try running nnupatcher again
    • If it gives an error, ensure that you do not have the update-blocking DNS installed
