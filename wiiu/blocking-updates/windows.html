Blocking Updates (Windows)

This step will allow you to block all future Wii U updates, without blocking access to the eShop like the DNS method did.
Blocking updates is not necessary, but recommended on 5.5.2 to prevent future updates breaking your Haxchi/CBHC setup.
If you do not have Haxchi/CBHC, it is recommended that you install it now.
If you are on 5.5.1 with Haxchi/CBHC, it is recommended to update to 5.5.2.

Instructions

  1. Download the latest release of Update Blocker Wii U
  2. Unzip the .zip file and open the folder Release
  3. Run the program ConsoleApplication1.exe
    • Make sure you're running FTPiiU_everywhere
  4. Read the text and press any key to continue
  5. Type in your Wii U's IP Address
  6. Type in 1 and then press Enter
  7. Exit out of FTPiiU_everywhere on your Wii U and return to the Home Menu
    • If you're on 5.5.2, ignore the following steps
  8. Open System Settings and navigate to System Update
  9. Update your system
    • If it fails instantly, congratulations, you successfully blocked updates
    • If it even makes the SLIGHTEST BIT OF PROGRESS, retry and see where you went wrong
    • If it tells you you're on the latest system firmware, you ignored my earlier instructions >:c
