Installing Haxchi

Haxchi is a Wii U exploit which allows you to run Homebrew from your Wii U on versions 5.5.1 and 5.5.2 with an 100% success rate. It does not require you to have internet access. We will later use Haxchi to launch Wii U CFW on boot. We will also install the Homebrew Launcher and the Homebrew App Store, so you can download homebrew easily.

Installing Haxchi

  1. Download Homebrew Launcher v1.4
  2. Download the latest release of Haxchi
  3. Extract the contents of the Haxchi .zip to the root of your SD card
  4. Extract the contents of the Homebrew Launcher .zip to the root of your SD card
  5. Insert the SD card into your Wii U
  6. Open the eShop and download one of these games
    • We recommend Brain Age, as it's the cheapest
    • A pirated copy won't work
    • If you own one of these games already, use that
    • If you're on 5.5.1 and don't want to update, use nnu-patcher to access the eShop
    • Make sure it's installed on your internal storage, not a USB
  7. Open up the Browser and navigate to cfw.guide/go
  8. Launch the Haxchi Installer
  9. Select the DS virtual onsole title you want to install Haxchi on
  10. Read the warning and then press A to install
  11. Go back to the Home Menu and launch Haxchi
    • If you can't see it or it doesn't work properly, try again

You should now reboot into a patched sysNAND

You can use Haxchi to launch many other homebrews by editing the haxchi/config.txt file too. The correct format is button=path, for example: x=wiiu/apps/homebrew_launcher/homebrew_launcher.elf. This would launch the Homebrew Launcher if the X button is held down while Haxchi is launched.

Installing Homebrew Titles

  1. Download the latest release of the Homebrew Channel (the channel .zip file)
  2. Download WUP Installer GX2
  3. Download the WUP Installer GX2 Channel
  4. Download Homebrew App Store v1.5
  5. Create a new folder in the root of your SD card named install
  6. Create a new folder in the install named hbc
  7. Extract the contents of the Homebrew App Store .zip to the wiiu folder on your SD card
  8. Extract the contents of the WUP Installer GX2 .zip to the root of your SD card
  9. Extract the contents of the Homebrew Channel .zip to the install/hbc folder on your SD card
  10. Extract the WUP_Installer_GX2 folder from the WUP Installer GX2 Channel .zip to the install folder on your SD card
  11. Launch the Haxchi title to reboot into a patched sysNAND
  12. Launch the Haxchi title again, but hold down A while launching it
  13. Launch WUP Installer GX2
  14. Select Homebrew Channel > Install > Yes > NAND
  15. Select WUP Installer GX2 > Install > Yes > NAND

You should now be able to launch the Homebrew Channel and WUP Installer GX2 from your Home Menu. You'll also be able to launch the Homebrew App Store from the Homebrew Channel to download whatever homebrew you want.
