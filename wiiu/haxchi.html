Installing Haxchi

Haxchi is an exploit which allows you to launch the Homebrew Launcher directly from your Home Menu
You won't need to use the browser exploit after this, just launch Haxchi from the Home Menu. It's as simple as that.

Installing Haxchi requires a legitimately installed DS Virtual Console title!
A legitimately installed title is a game which has been bought from the eShop.
If the title is not legitimately installed, you will not be able to launch it on OFW (Without CFW).
Haxchi can not be used on OFW without buying a title. Anyone who says otherwise is lying.

Preparations

  1. Download the latest release of Haxchi
  2. Extract the contents of the Haxchi ".zip" to the root of your Wii U's SD Card
  3. Put the SD Card into your Wii U

Instructions

  1. Open the eShop on your Wii U
  2. Download and install one of the titles listed above if you haven't already
  3. Return to the Home Menu and open the Internet Browser
  4. Navigate to cfw.guide/go to launch the Homebrew Launcher
    • Alternatively go to loadiine.ovh to launch the exploit
  5. Launch the Haxchi Installer
  6. Select the DS virtual console title you wish to install Haxchi to
  7. Read the warning that appears
  8. Press A to install
  9. Go back to the Home Menu when this has completed
  10. Launch the new "Haxchi" title

The console will now reboot into a patched sysNAND.
From here, you'll be able to install homebrew applications onto your Home Menu
