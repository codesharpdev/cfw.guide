Updating to 5.5.2

To exploit your Wii U, we'll be using some software called Haxchi. This unforunately is only installable on versions 5.5.1 and 5.5.2. Luckily, 5.5.2 is the latest version, so we can easily just update through System Settings

Instructions

  1. Open System Settings from the Home Menu
  2. Scroll right and then select System Update
  3. Update your system
    • Make sure you have update blocking disabled
