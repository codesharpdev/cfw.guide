WiiU Guide

Homebrew Launcher (Channel)

This step will enable you to launch the Homebrew Launcher from your Home Menu.

Preparations

  1. Download the Homebrew Launcher (Channel)
  2. Download WUP Installer GX2 (Homebrew Launcher)
  3. Download WUP Installer GX2 (Channel)
  4. Copy and merge the contents of the WUP Installer GX2 (Homebrew Launcher) ".zip" to the root of your SD card
  5. Copy the contents of the Homebrew Launcher (Channel) ".zip" to the directory "SD:install/hbc
  6. Copy the WUP Installer GX2 (Channel) ".zip" to the directory "SD:install"
  7. Put the SD Card into your Wii U

Instructions

  1. Launch the Homebrew Launcher
    • CBHC users, do this by pressing the Home Button when the CBHC screen appears on boot and select "Boot Homebrew Launcher"
    • Haxchi users, do this by launching Haxchi while holding A
    • Browserhax users, do this by opening the Wii U browser and going to cfw.guide/go
  2. Launch WUP Installer GX2
    • If it crashes, just reboot and try again.
  3. Select the Homebrew Channel
  4. Select "Install > Yes > NAND"
  5. Select WUP Installer GX2
  6. Select "Install > Yes"
    • If you are using a USB device to store games on your Wii U, select USB as the destination
    • If you are not, select NAND as the destination
  7. Press Home to exit and go back to the Home Menu

Both HBL and WUP Installer GX2 should appear on your Home Menu.
You can launch them in patched sysNAND, which should automatically boot with CBHC.
