Installing Mocha

Mocha is a CFW which will allow you to patch signature checks without Haxchi/CBHC.
You will be required to run it on every reboot, unless you have CBHC installed.

Preparations

  1. Download the Homebrew Launcher
  2. Download the Homebrew App Store
  3. Download the latest release of Mocha CFW
  4. Create the directory SD:wiiu/apps/mocha and extract the contents of the Mocha CFW .zip into it
  5. Extract the contents of the Homebrew Launcher .zip to the root of your Wii U's SD Card
  6. Extract the contents of the Homebrew App Store .zip to the SD:wiiu directory on your Wii U's SD Card
  7. Put the SD Card into your Wii U

Instructions

  1. Launch the Homebrew Launcher by going to cfw.guide/go on your Wii U
    • Alternatively, use loadiine.ovh
  2. Use the D-Pad to navigate through and set the following settings
    • Config view mode: default
    • Skip this menu on launch: on
    • Show launch image: on
    • Don‚Äôt relaunch OS: off
    • Launch System Menu: on
    • RedNAND: off
  3. Press A to confirm and exit

The console will now reboot into a patched sysNAND.
From here, you'll be able to install homebrew applications onto your Home Menu
