Updating to 5.5.1 (Hax)

Well, this is technically an update to 5.5.2, but I named it this not to confuse people.
In reality it just updates to 5.5.2 but keeps the 5.5.1 browser exploit, so you can easily launch Homebrew.

You will need to use nnu-patcher to use eShop/update applications, and you'll need to make sure you're blocking updates to make sure that everything goes well.

Preparations

  • Download Java if you haven't already
  • Download OSTitleDownloader
  • Download WUPInstaller (No blacklist)
  • Download the Wii U common.key
    • I cannot link it here, as it is against copyright law
  • Download your correct region and firmware title list

5.4.0 5.4.0 5.4.0
5.3.2 5.3.2 5.3.2

WARNING: This next part could potentially brick your Wii U if you do not follow the instructions correctly.
Make sure to read all instructions through before proceeding.
I will not take responsibility for any damage done to your system.

Instructions

  1. Install Java if you haven't already
  2. Put OSTitleDownloader.jar, LIST.txt and common.key into the same directory
  3. Open a terminal/command prompt in that directory and type java -jar titledownloader.jar -l LIST.txt
  4. Once it's finished, copy the install folder to root of your SD Card
  5. Extract the contents of wupinstall-noblacklist.zip to the root of your SD card
  6. Run the Homebrew Launcher by typing in cfw.guide/go into the URL bar
    • Alternatively use loadiine.ovh
  7. Run WUPInstaller - No Blacklist
  8. Install all titles EXCEPT OSv10 (000500101000400A) to NAND
  9. Now, you can install OSv10 (000500101000400A) to NAND
  10. The console will freeze. Reboot

Your Wii U should now be on 5.5.2 with a 5.5.1 browser.

You will be able to go online in games, however to use the eShop and to update games, you must use nnu-patcher
It is also highly recommended that you now block system updates
