LEGO Star Wars - Return of the Jodi

This step will help you install Homebrew using LEGO Star Wars.

Preparations

  1. Download Return of the Jodi
  2. Download Hackmii Installer
  3. Extract the private folder from the Return of the Jodi .zip to the root of your SD Card
  4. Extract the boot.elf file from the Hackmii Installer .zip to the root of your SD Card
  5. Turn on your Wii U and go to the vWii Menu
  6. After it has loaded, put in your SD Card
  7. Go to Wii Options > Data Management > Save Data > Wii
  8. Go to SD Card and select the Return of the Jodi savegame which corresponds to your game region
  9. Copy the savegame to the vWii

Instructions

  1. Boot your LEGO Star Wars disc
  2. Load the saved game you just copied
  3. Go to the bar on the right
  4. Choose the character named Return of the Jodi and press A
  5. Hackmii Installer should now load

Installing the Homebrew Channel

  1. You will be informed that you should not have paid for this software
    • We didn't, so that's good
  2. After a few seconds, you will be able to proceed
  3. Install the Homebrew Channel
    • Not BootMii, you can't install that on a vWii
