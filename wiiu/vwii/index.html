Get Started

What is the "vWii"?

The vWii is the Wii U's "virtual Wii". The vWii can run Wii and Gamecube games on the Wii U. This also means we can install homebrew on it, which is what this guide will show you how to do.

Running homebrew on the vWii is 100% free using the Wii U's browser.

Why should I hack my vWii?

Hacking your vWii can give you access to such benefits as

  • Playing Gamecube games on your Wii U
  • Dump your discs to a digital format, and play them off a USB Memory Device
  • Play games for older systems with various emulators, using RetroArch or other standalone emulators
  • Launch many homebrew titles and games on your vWii

What is the hacking process?

This guide will:

  • Install the Homebrew Channel to your Home Menu
  • Dump important info from the vWii (optional but recommended)
  • Install cIOS
  • Install a launcher to play backups of vWii Games
  • Install Nintendon't to play Gamecube games
  • Install Emulators to play older games
  • Install Forwarder Channels to launch Homebrew from the Home Menu
  • Install Virtual Console or eShop games

Most of these are optional, however you can install them if you want
