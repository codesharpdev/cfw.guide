Game Dumping

Dump your physical (or digital) games into digital format. No need to carry around your games anymore!
Make sure you have sufficient space on your memory card/microSD card

  1. Open the Homebrew Browser App
    • Skip to step 5 if you already have MaiDumpTool installed
  2. Select "Utilities" at the top
  3. Scroll down to "MaiDumpTool (English)"
  4. Select it and install the application
    • Direct Download Link
  5. Open MaiDumpTool from the Home Menu
    • Remember that while using MaiDumpTool, the O button confirms, and the X button cancels, similarly to Japanese models
  6. Select "Extract/Decrypt Content"
  7. Select the game you wish to dump
  8. Select "Extract game and decrypt eboot"
  9. Your game has now been dumped!
