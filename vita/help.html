Help

If you're having any issues following the guide, check the solutions here before asking on our Discord Server for help.

Error C2-12828 when launching HENkaku

  • Make sure you have a strong Wi-Fi signal
  • Delete your cookies in the PS Vita Browser
    • Make sure that after each step, you keep trying for a bit before moving on
  • Turn your Vita off, wait a few minutes, turn it back on again and then try again
  • Start the Vita in Safe mode and select "Rebuild Database"
    • Turn your Vita off
    • Press the "R" button + PS button + Power button simultaneously for 5 seconds while the Vita is off
    • Select Rebuild Database
  • Reset the Wi-Fi Settings
  • If all that fails, try formatting the memory card
