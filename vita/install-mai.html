Backup Installing

Install MaiDumpTool Backups
This guide is for installing MaiDumpTool backups, not Vitamin/VPK backups.
MaiDumpTool backups are superior, as you do not need twice the capacity of the game, as you do with Vitamin.
If you wish to install Vitamin dumps, simply select them in VitaShell and install them.

  1. Open the Homebrew Browser App
    • Skip to step 5 if you already have MaiDumpTool installed
  2. Select "Utilities" at the top
  3. Scroll down to "MaiDumpTool (English)"
  4. Select it and install the application
    • Direct Download Link
  5. Put your decrypted MaiDumpTool backup in the "ux0:mai" folder
    • Example: ux0:mai/PCSE00120/
  6. Open MaiDumpTool
    • Remember that while using MaiDumpTool, the O button confirms, and the X button cancels, similarly to Japanese models
  7. Select "Install game from folder"
    • Select "Install Patch/DLC" if installing updates/DLC
  8. Follow the on-screen instructions to install
