Finalising Setup

This step will finish the installation and set you ready for homebrew

Download Enabler v3

  1. Open the Web Browser on your PS Vita
  2. Visit this page on your Vita and click this link to download Download Enabler v3
  3. Download the file
  4. Go back to the Home Menu and open MolecularShell
  5. Navigate to "ux0:video/aa
    • Not necessarily "aa", it could be any two letters, e.g. "hg", "cf", etc
  6. Rename "de3.mp4" to "de3.suprx"
  7. Press Triangle and select "Copy" to copy this file
  8. Navigate to "ux0:tai" and paste it there
  9. Open "config.txt"
  10. Add the following to the file:
  11. Add the following to the file:
    *main
    ux0:tai/de3.suprx
  12. Exit and save changes
  13. Reboot your console

Installing Homebrew

  1. Open this page up on your Vita (vitahax.com/finalising-setup)
  2. Download both of the following files
    • VitaShell
    • Vita Homebrew Browser
  3. Open MolecularShell
  4. Navigate to "ux0:download"
  5. Install "VitaShell.vpk"
  6. Install "VitaHBBrowser.vpk
    • If the install fails, open VitaShell and install it through there
  7. Open the Settings App
  8. Select "HENkaku Settings"
  9. Uncheck "Enable Unsafe Homebrew"
  10. Open the Homebrew Browser app
  11. Browse through the list for something you want: your memory card is the limit!
