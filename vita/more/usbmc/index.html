USBMC

USBMC, just like SD2Vita, is a solution to the expensive Vita memory cards. It allows you to use USB memory with a PSTV, or a microSD card with PSVD boards on 3G Vita 1000s.

Instructions

  1. Download the latest release of usbmc
  2. If you haven't already, move your ux0:tai folder to ur0:tai and delete the ux0:tai
  3. Install the USBMC .vpk you downloaded
  4. Open the USBMC Installer application from the home menu
  5. Press X to install the drivers and reboot
  6. Enter the USBMC Installer application again
  7. Press X to copy the files to your USB or microSD Card
    • Make sure you have inserted your USB memory into the PSTV first
