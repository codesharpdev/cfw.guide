Installing Adrenaline

This step will install Adrenaline on your Vita, so you can play PSP and PS1 Games & Homebrew
Adrenaline no longer requires activation, so you can install it on any device
If you installed Adrenaline previously, you must uninstall it before updating to Adrenaline v6

Instructions

  1. Open the Homebrew Browser App
  2. Scroll down to Adrenaline by TheFloW
  3. Select it and install the application
    • Download (GitHub)
  4. Scroll down to "PSP Homebrew Browser"
  5. Select it and install the application
    • Direct Download Link
  6. Open Adrenaline
  7. Press X to download the PSP 6.61 firmware
    • This will take a while
    • The application will automatically close after it's finished
    • If you previously installed Adrenaline, move ux0:adrenaline/661.PBP to ux0:app/PSPEMUCFW/
    • Afterwards, delete the ux0:adrenaline directory
  8. Open the application again
    • If the Adrenaline application kicks you back to the Home Menu automatically, just open the application again
    • If it still isn't doing anything, reboot, and open the application twice
    • This is an issue which will be fixed by the end of the guide
  9. Press X to install the PSP 6.61 firmware
    • This will take a while
  10. Press X to boot the PSP XMB
  11. Setup the PSP
  12. Hold the PS button and select Settings
  13. Select Exit PspEmu Application

Install PSP Homebrew with the PSP Homebrew Browser
It can also be installed by being placed into the "ux0:pspemu/PSP/GAME" directory.
Place ISO game files in the "ux0:pspemu/ISO" directory.

Fixing Adrenaline Launching (optional)

Adrenaline v6 requires Adrenaline to be launched twice after reboot. If this bothers you, you can fix it

  1. Open VitaShell and navigate to ur0:tai/config.txt
    • If you have your config at ux0, move it to ur0
  2. Write the following changes:
    3. *KERNEL
    ux0:app/PSPEMUCFW/sce_module/adrenaline_kernel.skprx
  3. Reboot and see if the changes have been made
