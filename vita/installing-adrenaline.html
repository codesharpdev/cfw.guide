Installing Adrenaline

This step will install Adrenaline on your Vita, so you can play PSP and PS1 Games & Homebrew

Adrenaline requires an activated Vita to install.
If you do not have an activated Vita, you cannot install Adrenaline unless you have a legit PSP/PS Mini game installed.

Instructions

  1. Open the Homebrew Browser App
  2. Select "Utilities" at the top
  3. Scroll down to "Adrenaline EasyInstaller"
  4. Select it and install the application
    • Direct Download Link
  5. Scroll down to "PSP Homebrew Browser"
  6. Select it and install the application
    • Direct Download Link
  7. Open Adrenaline EasyInstaller
  8. Select "3. Advanced Options"
  9. Select "Install a small PSP Basegame if needed"
    • Not required if you already have a PSP game installed
  10. Wait for it to finish and press X to reboot
  11. Once you have rebooted, you should notice a new PSP game on your Home Menu. Do not open it, it won't work yet.
  12. Open up Adrenaline EasyInstaller again
  13. Select "1. Adrenaline eCustomFirmware"
  14. Select "Install the latest Adrenaline eCFW release"
  15. Select the PSP Basegame you just installed
    • It should be "NPEZ00104 - BIGHEADSNOOKER"
  16. Confirm the installation and wait for it to finish. This will take a while.
  17. Once it's finished, press X to reboot
  18. Adrenaline should now be installed

Install PSP Homebrew with the PSP Homebrew Browser
