Vita Guide

Installing Adrenaline

This step will install Adrenaline on your Vita, so you can play PSP and PS1 Games & Homebrew

Instructions

  • Open the Homebrew Browser App
  • Select "Utilities" at the top
  • Scroll down to "Adrenaline EasyInstaller"
  • Select it and install the application
  • Scroll down to "PSP Homebrew Browser"
  • Select it and install the application
  • Open Adrenaline EasyInstaller
  • Select "3. Advanced Options"
  • Select "Install a small PSP Basegame if needed"
  • Not required if you already have a PSP game installed
  • Wait for it to finish and press X to reboot
  • Once you have rebooted, you should notice a new PSP game on your Home Menu. Do not open it, it won't work yet.
  • Open up Adrenaline EasyInstaller again
  • Select "1. Adrenaline eCustomFirmware"
  • Select "Install the latest Adrenaline eCFW release"
  • Select the PSP Basegame you just installed
  • It should be "NPEZ00104 - BIGHEADSNOOKER"
  • Confirm the installation and wait for it to finish. This will take a while.
  • Once it's finished, press X to reboot
  • Adrenaline should now be installed

    • Install PSP Homebrew with the PSP Homebrew Browser
    It can also be installed by being placed into the "ux0:pspemu/PSP/GAME" directory.
    Place ISO game files in the "ux0:pspemu/ISO" directory.
