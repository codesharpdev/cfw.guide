3.60 Update

This step will update your Vita to 3.60, as HENkaku only works on this version.
Make sure that you are connected to the internet before doing this.

Simple Instructions

  • Set your DNS to 212.47.229.76
  • Update your system from Wi-Fi
  • Proceed only if the message says "A new version of the system software is available. Firmware 3.60 (変革 Compatible)”
  • Update your system to 3.60

    • Proceed to Installing HENkaku

    If you don't know how to do this, follow these instructions below for a more detailed version

    Detailed Instructions

  • Open the Settings app
  • Select "Network"
  • Select "Wi-Fi Settings"
  • Make sure you are connected to the internet.
  • Select "Advanced Settings"
  • Make sure "Proxy Server" is set to "Do Not Use"
  • Select "DNS Settings"
  • Choose "Manual"
  • Select "Primmary DNS"
  • Enter "212.47.229.76
  • Leave "Secondary DNS" blank
  • Go back to the main Settings screen
  • Select "System Update" and then "Update Using Wi-Fi"
  • If you are given an error, it means that either your Vita is either on 3.60, 3.61, 3.63 or 3.65. If you are on 3.61 or above, you cannot install HENkaku
  • If all goes well, you will see a pop-up saying that there is a new update
  • MAKE SURE THE MESSAGE SAYS "3.60 (変革 Compatible)"!!! If it does not, DO NOT PROCEED and make sure you followed all these steps exactly
  • If the message says "3.60", go ahead and update your system
  • You will now update the 3.60

    • Proceed to Installing HENkaku
