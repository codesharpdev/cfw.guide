3.60 Update

This step will update your Vita to 3.60, as HENkaku only works on this version.
Make sure that you are connected to the internet before doing this.

Instructions

  1. Open the Settings app
  2. Select "Network"
  3. Select "Wi-Fi Settings"
  4. Make sure you are connected to the internet.
  5. Select "Advanced Settings"
  6. Make sure "Proxy Server" is set to "Do Not Use"
  7. Select "DNS Settings"
  8. Choose "Manual"
  9. Select "Primmary DNS"
  10. Enter "212.47.229.76
  11. Leave "Secondary DNS" blank
  12. Go back to the main Settings screen
  13. Select "System Update" and then "Update Using Wi-Fi"
  14. If you are given an error, it means that either your Vita is either on 3.60, 3.61, 3.63 or 3.65. If you are on 3.61 or above, you cannot install HENkaku
  15. If all goes well, you will see a pop-up saying that there is a new update
  16. MAKE SURE THE MESSAGE SAYS "3.60 (変革 Compatible)"!!! If it does not, DO NOT PROCEED and make sure you followed all these steps exactly
  17. If the message says "3.60", go ahead and update your system
  18. You will now update the 3.60

Proceed to Installing HENkaku
